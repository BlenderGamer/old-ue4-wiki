Unreal Tournament Release Notes

Version notes for build released 7/12/2017


Major Changes This Build:

  • Performance improvements .
    • Fixed static meshes using LOD distance scaling.
    • Underland optimizations to landscape.
    • Added new mesh LODs.
    • Increased medium texture pool size.
  • Fixed memory allocation crashes on Mac and Linux.
  • Fixed disabled ranked leagues being available in UI.
  • Fixed cases where new build notice was not displayed for players.
  • Don't time out the render thread if taking a highres screenshot.
  • Misc. rare crash fixes.
  • Fixed dual enforcer lighting issues.
  • Fixed animation glitch when switching from out of ammo weapon to Stinger.
  • Show line up even when just one player.
